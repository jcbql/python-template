# Python template
This is a template for python projects with [devcontainers](https://code.visualstudio.com/docs/devcontainers/containers) and [uv](https://docs.astral.sh/uv/).

> [!IMPORTANT]
> Remember to rename `template` in [`pyproject.toml`](pyproject.toml), [`.devcontainer/Dockerfile`](.devcontainer/Dockerfile), and [`.devcontainer/devcontainer.json`](.devcontainer/devcontainer.json).
